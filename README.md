## 環境構築手順
1.GitHubよりダウンロード

```
$ git clone https://github.com/TomokoMineyuki/resepj.git
```
2.マイグレーションの実行

```
$ php artisan migrate
```
3.シーダーの実行

```
php artisan db:seed
```
